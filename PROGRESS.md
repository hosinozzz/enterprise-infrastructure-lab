## Status Definition

| Status | Meaning |
| --- | --- |
| Not Started | 未着手 |
| In Progress | Hands-on実施中、または記録作成中 |
| Done | 実施済みで記録も完了 |
| Blocked | 前提条件不足、環境不足、権限不足などで停止中 |

## Documentation Completion Criteria

技術記録を `Done` にする条件は以下です。

- 実施した環境が記載されている
- 実施した日付が記載されている
- 実際に行った操作が記載されている
- 確認結果が記載されている
- 発生した問題があれば記載されている
- 推測と事実が混在していない
