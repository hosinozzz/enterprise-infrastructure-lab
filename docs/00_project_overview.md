# Project Overview

## 目的

Enterprise Infrastructure Lab は、Oracle Database 19c の構築学習および Oracle Database Silver DBA 資格取得を目的としたハンズオン構築プロジェクトです。

Oracle Database のインストール、OS 前提設定、Oracle ユーザー管理、Oracle Home 構成、GUI 環境、DBCA、Listener、バックアップ・リカバリ、移行検証を段階的に学習します。

## GitHub Portfolio としての位置付け

本リポジトリは、将来的に GitHub Portfolio として公開することを想定しています。

そのため、単なる作業メモではなく、以下を意識して記録します。

- 目的が分かること
- 実施内容と確認結果が分かること
- 未検証事項が分離されていること
- トラブルシューティングの経緯が残っていること
- 再現可能な構成で整理されていること
